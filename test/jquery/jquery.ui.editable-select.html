Basic example

It transforms this
Into this

Code

$('#basic').editableSelect();

Effects

Default
Slide
Fade

Code

$('#default').editableSelect({ effects: 'default' });
$('#slide').editableSelect({ effects: 'slide' });
$('#fade').editableSelect({ effects: 'fade' });

Dialog

Open modal dialog

Modal title

Code

$('#appendTo').editableSelect({
    appendTo: '.modal-body'
});

No filtering

Code

$('#no-filter').editableSelect({ filter: false });

HTML support

Code

$('#html').editableSelect();

Listen changes

Last selected:

Code

$('#select')
    .editableSelect()
    .on('select.editable-select', function (e, li) {
        $('#last-selected').html(
            li.val() + '. ' + li.text()
        );
    });
